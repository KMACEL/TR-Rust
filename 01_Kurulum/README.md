# TR-Rust

## Yükleme

```bash
curl https://sh.rustup.rs -sSf | sh
```

## Güncelleme

```bash
rustup update
```

## VSCode Kurma

ctrl+p

```bash
ext install rust-lang.rust
```

## Proje oluşturma

```bash
cargo new merhaba-dunya
```

## Proje Çalıştırma

```bash
cargo run
```

## Projeyi Derleme

```bash
cargo build
```

## Release Derleme

```bash
cargo build --release
```

## Projeyi Kontrol Etme

```bash
cargo check
```

## Cargo'suz Derleme

```bash
rustc main.rs
```

## Yerel Doküman

```bash
rustup doc
```

## Rust Düzenleme Aracını Yükleme

```bash
rustup component add rustfmt --toolchain stable-x86_64-unknown-linux-gnu
```
